Insertando imágenes en documentos HTML

Las primeras páginas web no tenían imágenes y se limitaban a monstrar textos porque el ancho de banda era muy pequeño. En la actualidad, la mayoría de las páginas de internet están inlustradas con imágenes. El diseño de las páginas web en HTML permite insertar imágnes fácilmente.

Esta es la primera imagen que inserto mi página web utilizando una ordern HTML.

En este caso se trata de una foto que he consegido en una web donde hay muchas imágenes para descargar.
